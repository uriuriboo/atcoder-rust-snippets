# atcoder-cpp-snippets README

## Features

This is a Rust snippet for AtCoder.

| Prefix   | Description       |
|----------|-------------------|
| tmplt    | Template          |
| oans     | Output ans        |
| oY       | Output Yes        |
| oN       | Output No         |
| pbit     | Bit search        |
| bins     | Binary search     |
| for0     | Loop from 0       |
| ffor     | Double loop       |


**Enjoy!**
